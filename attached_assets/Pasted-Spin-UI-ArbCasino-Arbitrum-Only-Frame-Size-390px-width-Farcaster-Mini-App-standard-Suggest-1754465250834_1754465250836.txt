🌀 Animation Ideas
1. Spin Reel Animation (Inspired by slot machines)
Each icon slot scrolls vertically.

Animate spin using framer-motion or @react-spring/web.

Duration: 2 seconds per spin.

Randomly select icons like 🎯, 🐸, 🪙, 💀, 🌈, 🍌.

2. Winning Flash Animation
If user wins:

Animate flashing “YOU WON!” text

Show reward token icon (e.g., PEPE frog emoji or image)

Fireworks burst via lottie-react animation

3. Button Feedback
On spin press:

Button scales down slightly → back to normal

Spinner sound (optional on mute)

🎨 Style Suggestions
Color Scheme
Use Arbitrum’s Blue: #28A0F0

Contrasting colors:

Background: #0B0F1C

Accent: neon green / bright yellow

Font
Use bold retro or arcade-style font for SPIN text.

Example: Press Start 2P (Google Fonts)

Normal text: Inter, Poppins, or DM Sans

Icons
Use SVG or PNG for each icon (e.g., slot emojis)

Consistent size: 64x64px

Add bounce animation to selected icons
