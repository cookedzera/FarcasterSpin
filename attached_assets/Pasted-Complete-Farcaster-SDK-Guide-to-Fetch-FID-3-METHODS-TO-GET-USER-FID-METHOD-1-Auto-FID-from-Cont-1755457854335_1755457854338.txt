METHOD 1: Auto FID from Context (EASIEST - NO AUTH REQUIRED)
Official Docs:

https://miniapps.farcaster.xyz/docs/sdk/context

SDK Installation:
html<script src="https://unpkg.com/@farcaster/miniapp-sdk@latest/dist/index.js"></script>
Get FID Automatically:
javascriptimport { sdk } from '@farcaster/miniapp-sdk';

// Initialize SDK
await sdk.init();

// FID is automatically provided in context!
const userFID = sdk.context.user.fid;
const userName = sdk.context.user.displayName || sdk.context.user.username;

console.log('User FID:', userFID);
console.log('User Name:', userName);

// Example output:
// User FID: 6841  
// User Name: Tony D'Addeo

METHOD 2: Quick Auth (AUTHENTICATED FID)
Official Docs:

https://miniapps.farcaster.xyz/docs/sdk/quick-auth

Get Authenticated FID:
javascriptimport { sdk } from '@farcaster/miniapp-sdk';

// Get authenticated session token
const { token } = await sdk.quickAuth.getToken();

// Make authenticated request to your backend
const response = await sdk.quickAuth.fetch(`${YOUR_BACKEND}/me`);
const user = await response.json();

console.log('Authenticated FID:', user.fid);
Backend Validation (Node.js/Hono):
javascriptimport { createClient } from '@farcaster/quick-auth';

const client = createClient();

// Verify JWT and extract FID
const payload = await client.verifyJwt({
  token: bearerToken,
  domain: 'your-domain.com'
});

const userFID = payload.sub; // FID is in the 'sub' field
console.log('Verified FID:', userFID);
